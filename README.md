# 電卓アプリ

ローカル環境で動作するシンプルな電卓アプリです。

## 機能

- 基本的な四則演算（加算、減算、乗算、除算）
- クリア機能（AC）
- 削除機能（DEL）
- 小数点対応
- キーボード操作対応
- レスポンシブデザイン

## 使い方

1. `index.html` をブラウザで開く
2. マウスまたはキーボードで操作

### キーボードショートカット

- **数字キー (0-9)**: 数字入力
- **+ - * /**: 演算子
- **Enter または =**: 計算実行
- **Backspace**: 一文字削除
- **Escape**: クリア
- **.**: 小数点
